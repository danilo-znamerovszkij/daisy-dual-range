daisy-dual-range

v0.1.0

Framework-agnostic dual-range slider with DaisyUI styling powered by noUiSlider

TypeScript DaisyUI noUiSlider

Installation

npm install daisy-dual-range nouislider
Then import the CSS and initialize: 
import 'nouislider/dist/nouislider.css';
import { createDaisyDualRange } from 'daisy-dual-range';
import 'daisy-dual-range/style.css';
